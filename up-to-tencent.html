Coding.net 如何升级至腾讯云开发者平台？

文章内容

  • 2 绑定原有CODING账号
  • 3 有多个CODING账号如何关联腾讯云账号？
  • 4 新的腾讯云开发者平台账号如何关联已有的CODING账号？
  • 5 已有项目绑定 Coding.net 账户
    • 5.1 通过删除项目绑定原 Coding.net 账户
    • 5.2 通过转让项目绑定原 Coding.net 账户

CODING 与腾讯云已达成战略合作，利用腾讯云强大的计算资源，携手为广大用户提供免费的研发（代码）管理产品服务– 腾讯云开发者平台（https://dev.tencent.com）。所有 Coding.net 用户均可通过以下升级指引升级至腾讯云开发者平台，即可获得永久免费、速度更快、体验更优的产品和服务。

我们确保，升级为腾讯云开发者平台账户仅给用户权益方面做了较大提升，在升级后原有的 Coding.net 主要代码托管、项目管理方面功能使用保持一致，您的项目、代码信息、功能使用均不会受到影响，产品和服务依然由 CODING 团队提供。（注：该升级针对 Coding.net 用户，对 CODING 企业版客户不受影响）

为了帮助您更快地升级账号，您可以根据本教程进行操作。

注册腾讯云账号

如果您已经有了腾讯云账户可以直接跳过这一步，如果没有的话建议您先创建腾讯云账户。

如果您没有腾讯云账号的话，在登录界面扫码后会自动跳转到腾讯云的注册流程，自动引导您注册腾讯云账号。

 
您可以关联已有的腾讯云账号或者注册新的腾讯云账号。
如果您要注册新的腾讯云账号的话，关注腾讯云助手即可。
如果您本身有腾讯云的账号，可以点击关联已有账号进去到关联腾讯云账号的步骤。

绑定原有CODING账号

注册成功腾讯云账户后（如果您已经有腾讯账号的话，您可以直接使用原本的腾讯云账号登录，无需注册，可以直接扫码微信登录），您可以访问 腾讯云开发者平台登录页来绑定原有的CODING账号（http://dev.tencent.com/login），假如您已经登录了腾讯云账号或者在第一步注册成功的话访问http://dev.tencent.com/login会自动跳转到下面这个界面：
请点击【关联 CODING 帐号】，关联成功后，原 CODING 账户内的项目资源将会自动迁移到开发者平台的帐号中。
 
点击【关联CODING用户】，并输入原本CODING的账号密码即可。
 
阅读提示并勾选【提示已读】，点击确认升级。

用户升级后，原 CODING 帐号无法在 coding.net 中继续使用，之前与 CODING 绑定的登录方式（github等）将会自动解绑，CODING 帐号内的项目资源将会被迁移到腾讯云开发者平台中。

原 CODING 中的冒泡，码币，商城，博客等功能将不在腾讯云开发者平台中继续提供，敬请理解。
CODING 付费会员升级后可联系 CODING 客服领取升级礼品。

 
当您进入到您熟悉的控制台时，恭喜您升级成功。原本的项目、任务、代码等都是不会丢失的。

有多个CODING账号如何关联腾讯云账号？

11月开始，腾讯云账号体系支持一个微信绑定多个腾讯云账号。而每个CODING账号可以绑定一个腾讯云账号，所以如果您有多个CODING账号的话，您可以注册同样数量的腾讯云账号并绑定微信（已有腾讯云账号如何绑定微信？）。
然后您可以按照上面的流程最开始的流程扫码后选择新的腾讯云账号登录。
将CODING的账号与腾讯云账号关联在一起。 所以如果您有多个CODING账号的话，您可以在一一关联后通过一个微信直接登录任意的CODING账号。

新的腾讯云开发者平台账号如何关联已有的CODING账号？

您可以在个人设置中，点击【关联CODING账号】绑定已有的CODING账号即可，绑定成功后关联CODING账号按钮会不再显示。需要注意的是，您的这个账号下不能创建过任何项目、任务。
温馨提示：
① 一个CODING账户只能关联一个腾讯云账号，升级完成后就无法重复升级了~
② 升级后项目地址会变动，记得修改已经clone到本地的项目。
③ 可以将多个账号里的项目手动迁移到一个账号中再进行升级。

已有项目绑定 Coding.net 账户

如果您本身是 Coding.net 的用户，在升级成腾讯云开发者平台账户过程中忘了绑定或者是自己重新注册了个腾讯云开发者平台并且希望使用当前腾讯云账号登录已经拥有的 Coding.net 账号，那么您可以通过关联 Coding.net 账户功能绑定原有的 Coding.net 账号，从而实现无缝升级。

正常情况下，您在您账号中没有任何项目的时候（如果只有自动生成的Demo项目的话，点击【确认关联】时，在当前页面出现一个提示框：“确认关联后，当前帐号下的【示例项目】将与当前帐号一同被锁定，您将无法登录使用。确认关联吗？”点击【确认】后立即关联，Demo项目是不会影响关联 Coding.net 账号这个操作的），可以在个人设置中看到关联 Coding.net 账号相关功能。但是如果您的账号下已经有了项目，点击【确认关联】时，在顶部提示：“您的项目不为空，请先清空或转让项目。”，您可以通过下面两个办法进行解决：

通过删除项目绑定原 Coding.net 账户

在项目的设置-项目设置中可以找到删除项目按钮，点击后输入密码即可。为了能够绑定 Coding.net 账号，您需要删除该腾讯云开发者平台下的所有项目。在删除前，请注意备份自己的数据。

删除所有项目后可以重新回到个人设置中绑定 Coding.net 账号。

通过转让项目绑定原 Coding.net 账户

您可以将项目先转给其它成员或者转给原本的 Coding.net 账号，这样等您绑定完 Coding.net 账号后，这个项目也就自动合并到当前腾讯云开发者平台账号中。

您可以现在项目的成员管理功能中点击【添加成员】按钮。

邀请您原本的 Coding.net 账号加入项目。

在项目设置中转让给刚刚添加成员：

转让成功后，您还需要退出该项目，这样您当前腾讯云开发者平台账号就没这个项目，通过这个方式可以将所有的当前账号下的所有项目都转给原本的 Coding.net 账号。

转让完所有项目后可以重新回到个人设置中绑定 Coding.net 账号。
