API网关

API 网关（API Gateway）是 API 托管服务。提供 API 的完整生命周期管理，包括创建、维护、发布、运行、下线等。您可使用 API Gateway 封装自身业务，将您的数据、业务逻辑或功能安全可靠的开放出来，用以实现自身系统集成、以及与合作伙伴的业务连接。

产品优势

优势 腾讯云 API 网关 开源网关
简单易用

API 网关用户可直接在管理控制台进行图形化配置，也可调用腾讯云 API 进行业务配置，无需自行搭建 API 网关部署所需要的设备。根据腾讯云提供的帮助文档，用户可快速搭建所需 API 网关服务

用户需要自建基础设施，通常涉及网络、管理系统、监控等一系列繁琐的配置。再将所需开源网关软件部署在物理服务器上，需要花费大量精力
易于维护

腾讯云提供对 API 网关的可视化监控，以及资源管理，租户隔离，权限控制等一系列的运维能力，将用户从繁重的运维中解放出来

需要用户自行管理资源，自行搭建管理监控等平台，运维工作繁琐
高可靠性

腾讯云提供 API 网关的集群部署，当您的网关节点出现故障时，可及时迁移，以实现高可靠性

单节点的硬件部署无法保证高可用性，多节点部署则成本高业务架构繁琐
成本低廉

API 网关仅在用户大量调用 API 后，按照调用次数及产生的流量收取低廉的费用

需要提前预付部署业务所需的物理机，带宽等大量资源费用，成本高昂
 

产品功能

  • API 生命周期管理
  • 认证与限流
  • API 监控
  • API 市场
API 生命周期管理
  • API 网关提供 API 创建、维护、发布、运行、下线等操作的全生命周期管理。
  • 并为您提供测试，预发布，发布等多种环境用以部署您的 API 。
  • 并提供便捷的 API 日常管理，包括版本管理，版本回滚等。节约因 API 管理而造成的工作量与人力。
认证与限流
  • 提供严格的认证服务，支持 HMAC 算法签名，用户使用 API 网关提供的密钥进行认证，没有被授予密钥的客户端无法调用您的 API ，可保护您 API 的安全性。
  • 限流功能可配置 API 的流控，您可按业务需要配置每秒支持调用 API 的次数，用以保障您业务的稳定性。
  • 前端支持 SSL 证书，保障前端传输的安全性。
API 监控

提供可视化的监控信息。可方便的查询运行中 API 的详细信息。方便用户的运维管理。为您对自身 API 的分析提供方便可靠的数据依据。以便 API 的后期迭代与维护，提高效率。目前提供的监控内容包括：

  • 请求数
  • 外网出流量
  • 响应时间
  • 前端错误数
  • 后端错误数
API 市场
  • 与 API 市场对接，用户可将自己 API 网关上配置的 API 作为产品发布在 API 市场中出售，将用户自身的产品功能，数据能力以 API 的形式开放出来开拓渠道，构建更加良好完整的生态。API 市场中，最终客户可针对业务选购不同的 API 服务，利用这些 API 服务，可加速和完善自身的开发进度。
  • API 市场可以为您解决 API 的计费，售卖等一系列问题。
 

应用场景

  • 微服务整合
  • 外部多端统一
  • 业务整合
  • 能力提供及售卖
微服务整合
  • 通过 API 网关，封装微服务的 REST API。
  • 均通过 API 网关 进行调用，享受 API 网关带来的统一鉴权，统一监控等好处。
  • 通过 API 网关，控制对外暴露的 API，减轻安全隐患。
外部多端统一
  • 通过 API 网关，一套后端服务来服务多个终端。
  • 通过协议适配和转换，满足不同终端对不同协议的需求。
  • 通过 API 网关，对不同终端分别授权，无需后端服务调整或配置。
业务整合
  • 通过 API 网关统一各业务子模块接口。
  • 聚合服务，可以使开发者更专注于业务迭代，而非底层资源及能力，解放开发者人力
能力提供及售卖
  • 以 API 形式提供能力，包含计算，数据，功能等各种能力
  • 利用市场，向开发者、合作伙伴、第三方，销售能力，赚取收入
