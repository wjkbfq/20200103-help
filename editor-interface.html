编辑器界面介绍

文章内容

  • 1 顶部菜单栏
  • 2 文件目录区
  • 3 代码编辑区
  • 4 功能区

编辑器即每个工作空间的工作界面，这是是 Cloud Studio 的主要操作界面，几乎所有的操作都在这里进行。一个工作区间包含 6 个部分：顶部菜单栏、文件目录区、代码编辑区、功能区、底部操作区和状态区，下面我们将逐一介绍。

顶部菜单栏

顶部菜单栏包含 Cloud Studio 的一些常用操作，如编辑器设置、文件操作、编辑操作、版本控制等等。在右侧还有腾讯云开发者平台的工作台入口。

文件目录区

文件目录区用于展示文件目录结构、当前操作文件路径，同时还包含一些常见文件操作，如上传、下载、重命名等等。

代码编辑区

代码编辑区即编写代码的区域，它支持所有主流语法高亮，同时支持 Java、HTML、TS、JS 的自动补全，对于 Markdown 和 HTML 文件还支持实时预览。代码编辑区支持多文件横向、纵向排列，你可以随意拖动位置来排列它们。

功能区

功能区是一些 Cloud Studio 特有的功能操作区，如运行环境、协同编辑、访问链接等。

底部操作区

提供终端、调试日志、项目网络等展示或操作区域，我们最常用的是终端窗口。

状态区

底部操作区显示当前工作空间的一些基本信息，如 workspace key、当前编辑位置、文件编码、语言、Git 仓库信息等，你也可以在这里切换编码、文件类型等。
