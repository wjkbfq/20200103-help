CODING 个人版
最适合研发团队的
项目协作工具

CODING 个人版包含了任务、Wiki、文件等功能。支持多成员协作，并且深度集成了代码仓库的操作与状态。

代码托管功能除了提供高性能远端 Git 仓库外，还支持代码审查、Bug 跟踪和一键部署等功能，

真正符合研发团队的工作模式。

高效的研发项目管理

基于 TODO list 模型，呈现详细的任务状态记录，看板模式辅助团队形成敏捷开发模式。

稳定可靠的代码仓库

高性能远端 Git 仓库，分布式计算和存储，无限制的云端计算资源。并提供保护分支、代码共享、代码收藏等功能。
