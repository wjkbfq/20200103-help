注册&登陆

进入腾讯开发者平台首页选择注册，填写好信息注册成功之后即可登陆个人腾讯开发者平台。

登陆可选择邮箱登陆或者微信扫码登陆。

通知中心

  • 登录腾讯云开发者平台，点击通知标志，既可以查看通知，待办，私信的消息。
  • 点击之后可以进入通知中心查看具体内容。

个人主页

点击个人头像可以进入控制台查看通知、动态、项目、待办等信息。

点击头像下拉菜单的【我的主页】可以查看自己参与的项目、活跃度、关注粉丝等信息。

个人设置

点击头像下拉菜单的【个人设置】可以设置自己的用户昵称、绑定手机号等基本信息。

修改头像

注册账户时会生成默认头像，若用户名称为全角字符，取最后两位；若用户名称为半角字符，取前面两位。

点击头像可进行修改头像。

修改密码

点击修改密码后出现修改密码弹窗，填写无误则可以修改密码。

修改邮箱

点击修改邮箱，填写未注册过的邮箱，会发送验证邮件，正确验证后修改成功。

访问令牌

设置访问令牌，对于用户的使用代码、文件等信息进行权限管理。

提醒设置

提醒设置是为 Coding 用户提供在coding平台上的相关信息提醒而设置的一个功能，您可以根据个人需求进行个性化设置。

进入个人主页后，点击『个人账户』——『提醒设置』即可在右边的选项中选择系统提醒或者邮件提醒完成设置。

配置SSH 公钥

腾讯开发者平台 支持使用 SSH 协议来访问 Git 仓库，提供账户 SSH 公钥和项目 SSH 公钥设置。 用户可以设置账户 SSH 公钥，获所有仓库的读写权限； 也可以在项目设置里面设置项目部署公钥，获取单个项目仓库的只读权限。有关 SSH 更多信息可参考 维基百科。

添加公钥后，您就可以在项目的代码页面点击 SSH 切换到 SSH 协议的 clone 地址，类似这样：

git@git.dev.tencent.com:wzw/leave-a-message.git

使用 SSH 协议来访问 Git 仓库，不需要每次链接都输入账号和密码。

注意：一个公钥只能认证一个用户，而一个用户却可以拥有多个公钥。

账户 SSH 公钥是跟用户账户关联的公钥，一旦设置，就拥有账户下所有项目仓库的读写权限。

生成公钥

打开命令行终端输入 ssh-keygen -t rsa -C “your_email@example.com”( 你的邮箱)，连续点击 Enter 键即可。

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
# Creates a new ssh key, using the provided email as a label
# Generating public/private rsa key pair.
Enter file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter]  // 推荐使用默认地址
Enter passphrase (empty for no passphrase):   //此处点击 Enter 键即可，也可以填写密码，填写密码后每次使用 SSH 方式推送代码时都会要求输入密码，由于这个 Key 也不是用于军事目的，所以也无需设置密码。

成功之后显示如下信息：

Your identification has been saved in /Users/you/.ssh/id_rsa.
# Your public key has been saved in /Users/you/.ssh/id_rsa.pub.
# The key fingerprint is:
# 01:0f:f4:3b:ca:85:d6:17:a1:7d:f0:68:9d:f0:a2:db your_email@example.com

添加公钥

  1. 在终端输入 open ~/.ssh，用文本编辑器打开『id_rsa.pub』文件，复制全部内容。
  2. 登录 Coding.net，进入『SSH 公钥』页面，点击『新增公钥』。
  3. 将第一步中复制的内容填写到『公钥内容』一栏，公钥名称可随意填写。
  4. 设定公钥有效期，可选择具体日期或设置永久有效。
  5. 点击『添加』，然后输入账户密码即可成功添加公钥。
  6. 完成后在命令行测试，首次建立链接会要求信任主机。

    注意： 同一个公钥只能绑定一个账户，Coding 暂时不支持同一公钥绑定多个账户。

进入账户 SSH 公钥设置页面。

添加公钥后，您就可以在项目的代码页面点击 SSH 切换到 SSH 协议的 clone 地址，类似这样：

git@git.dev.tencent.net:wzw/leave-a-message.git

使用 SSH 协议来访问 Git 仓库，不需要每次链接都输入账号和密码。

部署公钥

部署公钥是跟项目关联的公钥，设置后有该项目的仓库的只读权限，添加方式同账户SSH 公钥一致。
