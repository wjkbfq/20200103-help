助力开发

腾讯云开发者平台是腾讯云与 CODING 共同为开发者打造的云端工具平台，旨在为更多的开发者带去便捷、高效的

开发体验，提供包括需求管理、代码编写、代码管理、代码运行的整套系统。

CODING 个人版

CODING 个人版为开发者提供了高效便捷的开发工具，包括 Git/SVN 代码托管、Pages 服务、任务管理、文件 Wiki 管理等模块，是绝佳的个人代码管理工具。

Cloud Studio

Cloud Studio 是一款基于浏览器的云端 IDE，拥有不输于本地 IDE 的完整功能，并提供了强大的云端特性：实时保存、协同编辑、快速部署、预制环境快速切换等。

​CODING 企业版

CODING 企业版是专为企业/团队/组织打造的软件开发协作平台，提供全流程的 DevOps 工具，涵盖需求管理、代码管理，测试管理、构建物管理、部署管理、知识管理等模块。可以更好地管理项目成员，便捷而深入地把握开发进度，让开发流程高效可控。
