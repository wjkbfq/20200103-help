Cloud Studio 简介

Cloud Studio 是基于浏览器的集成式开发环境，为开发者提供了一个永不间断的云端工作站。在 Cloud Studio 写代码的体验以接近原生 IDE，同时也包含 Git 集成、插件系统、调试和终端等功能。除此之外，它还有很多基于云端特别的功能，让开发者开发编译起来更加便捷。

  • 一键环境切换：内置 Python、Java、Node.js 等多种环境，可以一键切换，摆脱繁琐的环境安装环节。

  • 一键部署：一键部署可以帮你快速部署动态（PHP，支持 MySQL）或静态网站，可用于部署 WordPress、CMS、Hexo 等应用。

  • 访问链接：为你的应用创建一个可以公开访问的链接，方便演示、调试和分享。

  • 在线协作：支持多人在线同时编写同一份代码文件，还可以在线聊天。可以帮助开发者进行面试、教学等。

工作空间

在 Cloud Studio 内，我们将一个项目称之为一个工作空间。一个工作空间是一个虚拟计算单元，我们为每个工作空间提供了独立的存储、计算资源以及开发环境。您可以在控制台（dashboard）管理您的工作空间。

目前每个用户最多有 5 个工作空间，并且只能同时运行一个工作空间，如果您需要打开另一个工作空间需要先关闭当前运行中的工作空间。不需要的工作空间可以删除，已删除的工作空间会展示在「最近删除」列表，保留 24 小时，在此之前您可以随时恢复。

Git 集成

Cloud Studio 内集成了 Git，进入某一个工作空间后，你可以使用交互式的界面进行代码提交、推送、合并分支等基本操作，甚至还可以进行变基、解决冲突等复杂工作。

插件系统

Cloud Studio 支持插件扩展，你可以安装别人开发的插件来增强使用体验，也可以自己开发插件发布到插件市场。你可以在插件管理页面管理自己的插件。

了解更多>

运行环境

Cloud Studio 内置常用的运行环境，如 PHP、Java、Ruby、Node.js、Machine Learning 等等，帮助您摆脱繁琐的环境安装环节。在创建工作空间时您可以选择一个运行环境，也可以在工作空间中切换、保存和重置环境。如果您在列表中没找到想要的环境，也可以先选择 Default 环境，再自行安装所需要的语言环境。

一键部署

一键部署可以帮你快速部署动态（PHP，支持 MySQL）或静态网站，可用于部署 WordPress、CMS、Vuepress、Hexo 等应用，同时支持域名绑定。

实时协同

Cloud Studio 支持多人同时编辑代码、实时查看协作者的编辑光标及聊天等功能，以满足教学、面试等协作场景。
