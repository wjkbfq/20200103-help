使用调试功能进行 Java debug

Cloud Studio 提供 Java Debug 功能，帮助开发者调试 Java 代码。这篇文章将介绍如何使用 Cloud Studio 进行 Java Debug。

首先，进入工作空间创建页面，选择「从模板创建」并选择「JavaDemo」，来创建一个 Java 工作空间。

Cloud Studio 会自动识别到这是 Java 项目并将语言服务设置为 Java，你可以点击左上角 logo ，进入设置-语言服务器来选择项目类型和源代码目录。

接下来，我们要在终端运行 mvn dependency:resolve 解析项目依赖。

现在，将鼠标移动到要调试的行左侧，会出现红色的断点，点击它则在此行打上断点。

打开右侧的「调试」面板，点击绿色的三角形按钮开始调试。

此时，Cloud Studio 会自动打开终端，将应用运行起来。

上一步我们已经启动了这个 Java 应用，现在选中右侧「访问链接」面板，点击上面的“＋”，创建一个访问链接。我们点击这个链接，去访问这个应用的网页。

此时，可以看到左侧断点处变成了一个小箭头，说明程序运行到这里暂停了，访问链接的网页也停在那里。

与此同时，在右侧的调试面板中出现了本地变量和调用栈列表，你就可以通过上面的一系列按钮操作和这些信息来调试代码，找出错误所在了。
