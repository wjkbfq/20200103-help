概述

腾讯云推出全新 FPGA 型实例。目前，该实例已提供三种实例规格，搭载 Xilinx Kintex UltraScale KU115 FPGA。

腾讯云和业界厂商有良好的关系，下一阶段也会推出基于 Intel 的 FPGA 硬件平台，方便开发者选择自己熟悉的开发模式，避免切换平台。

与其他云服务器 CVM 实例一样，您可以采用包年包月的计费模式为 FPGA 计算实例支付预付款。内测阶段，我们将为获取内测资格的用户免费提供 FPGA 实例。

您可以采用腾讯云提供的 FPGA 镜像进行部署和开发，目前腾讯云官方提供的 FPGA 镜像都是免费的，您可以调用基于深度学习的 Alexnet 模型的 FPGA 知识产权（IP）进行图片分类功能，加速您的产品应用。将来，您将可以在腾讯云服务市场上为其他客户无偿或有偿地提供您开发设计的 FPGA 镜像。

计算能力
硬件加速

FPGA 实例使用 FPGA 硬件加速器协同执行浮点数计算和图形处理等，借助 FPGA 实例，您可以轻松地使用 FPGA 为您的工作负载部署硬件加速，帮助您的应用程序降低处理时延并大幅提升处理能力。与仅使用 CPU 的服务器相比，应用程序运行速度最高可提升 20 倍。

可编程

FPGA 实例的 FPGA 是可编程的，您可以使用软件进行配置，灵活地更新和优化您的硬件加速，且无需重新设计任何硬件，非常适合深度学习、图形图像压缩处理、基因组学等其他 FPGA 计算工作负载。

机型配置

FPGA实例具有以下特征：

  • FX2 实例搭载 Xilinx Kintex UltraScale KU115 FPGA
  • 每块 FPGA 都具有 2 条 16 GiB 本地 DDR4 ECC 受保护内存，并配有专用 PCle x8 连接
  • 每块 FPGA 大约包含 130 万个逻辑单元以及 5520 个数字信号处理（DSP）引擎

深度学习模型

腾讯云 FPGA 实例尤其适用于对时间和效率要求比较高的高性能计算 (HPC) 应用程序。

随着移动互联网的发展，基于用户社交平台，用户正在创造庞大的图片集合且增长速度很快。为了增强图片分类检测的处理能力，降低图片检测成本，我们使用 FPGA 云服务器对深度学习模型中 CNN 算法的 Alexnet 模型进行加速计算。

实验表明，采用 Alexnet 模型对图像进行分类检测对比测试，在性能上，FPGA 云服务器处理性能是 CPU 云服务器的5倍。

实时图像压缩

常用的图片格式有 JPEG 格式、WEBP 格式等， WEBP 图片格式比 JPEG 图片格式存储空间小 30% 。为节省存储空间，降低传输流量，提升用户的图片下载体验，通常采用 WEBP 格式进行存储及传输分发。但 WEBP 压缩计算复杂度是 JPEG 压缩的10倍以上，采用 CPU 进行 WEBP 转码成本很高。为了增强图片转码能力，可使用 FPGA 云服务器进行加速，且不会影响视图片质量。F1 实例是满足这些应用程序要求的理想解决方案。

对JPEG 格式图片转成 WEBP 格式图片进行测试对比，测试图片大小为 853x640，FPGA 云服务器处理延时相比 CPU 服务器降低20倍，FPGA 云服务器处理性能是 CPU 服务器的6倍。
