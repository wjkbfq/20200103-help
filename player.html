产品简介

播放器 SDK ( Player+ ) ，基于腾讯云强大的后台能力与 AI 技术，提供视频点播和直播播放能力的强大播放载体。流畅稳定的播放性能，集广告植入、数据监测等功能于一身。覆盖多类应用场景，满足客户多样需求，让客户轻松聚焦于业务发展本身，畅享极速高清播放新体验。

在线视频

用户在观看在线视频时，能根据网络实际状况无缝自动切换观看清晰度，首屏秒开加低卡顿低延迟，尽享极速高清稳定观看体验，精彩镜头智能打点，不错过每一刻精彩时分。

游戏直播

基于游戏直播场景，在游戏空闲等待期智能插入闲时广告，增强品牌曝光，促进流量变现。提供多样式弹幕，加强用户间交流，支持各类主播与观众的互动应用，让观众更有带入感，引爆直播现场。

电子商务

基于点播或直播中的各类商品，提供商品详情展示及购买链接，促进用户购买转化。实时监测并统计用户观看行为数据，生成高质量分析结果，助力商业生态成长。

体育赛事

极速高清播放各大体育赛事，对各类精彩赛事的比分实时提供浮窗展现。低延迟低卡顿，让用户的观赛体验得到巨大提升。

在线教育

播放器 SDK 基于教育场景提供了交互式的白板功能，让师生能够进行实时的涂鸦互动；同时还支持对各类在线教育视频提供五分钟试看、缓存后再加密等功能，保护教育版权，充分尊重知识价值。

产品架构

播放器 SDK，卓越的播放体验

  • 除了支持点播、直播的基础播放能力之外，倍速播放、进度条打点、码率自适应、播放器小窗播放等功能超级播放器都一应俱全，为用户提供最卓越的播放体验。

强大底层技术支撑，播放更清晰、流畅

  • 以 P2P 技术为基础，播放器 SDK 通过 SDK 接入手机、OTT 等终端设备构建视频分发网络，充分利用闲置带宽，有效降低视频分发成本。
  • 腾讯云超分技术通过自研算法对视频内的画面进行降噪处理，极大优化了视频的清晰度，实现低清晰度视频的分辨率提升，提高了用户观看视频的体验。
  • 播放器 SDK 通过使用 H.265 解码技术，不仅提高了视频压缩转码效率，而且在降低复杂度的基础上支持传输更加清晰的视频。

多平台覆盖，播放器 SDK 快速接入

  • 覆盖了包括 Web、H5、iOS、Android 等多平台终端，播放器 SDK 提供了快速接入的 js 代码和 SDK，让您能够几步就完成接入和使用。
  • 配合 QQ 浏览器的 X5 内核，播放器 SDK 借助 MSE 标准实现了 Web / H5 的媒体源扩展，使播放器 SDK 能够在浏览器中实现强大的功能。

丰富的场景增值，让播放器不止于播放

  • 通过 AI 智能识别技术，使对应的植入内容与视频内容高度融合，实现广告 / 品牌 / 内容三者无缝衔接，有效提升您的品牌认知。
  • 播放器 SDK 支持广告的植入，同时腾讯云也提供了集成化的广告方案，能够让您快速地通过视频流量实现变现，敬请期待。
