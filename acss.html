账号安全服务 ACSS 简介

腾讯云账号安全服务（Account Security Service ，ACSS）是一套保障账号安全的服务，能够主动记录账号下的资源操作日志，帮助您监控您的用户活动和 API 使用情况。同时还可以基于日志数据提供实时分析，风险监测，异常告警的能力，极大地保障了您的账号安全。

提供账号日志记录服务
提供账号安全监测服务
提供潜在风险告警服务

账号安全服务的子产品

云审计

云审计是一项支持对您的腾讯云账号进行监管、合规性检查和操作审核的服务。提供记录日志、持续监控并保留云资源的操作日志的能力。

账号威胁发现

账号威胁发现为用户提供账号安全监测服务。基于腾讯云深度积累的威胁情报源、数据分析、机器学习能能力主动发现并上报威胁事件。

腾讯云账号安全服务的特性

风险识别

基于云审计（CloudAudit）的数据进行分析，并快速建立用户行为基准，实时帮助用户分析每件发生的 API 调用是否符合用户行为基准。依托于腾讯云多年积累的威胁情报源（例如：恶意IP列表）、机器学习、异常检测等资源以及技术，为用户账号完全保驾护航。

操作便捷

用户无需额外安装任何软件，仅需登录腾讯云控制台开通服务，即可及时记录、存储腾讯云账户的操作行为记录以及 API 调用记录、持续分析监控用户行为，并及时反馈威胁事件。

合规性审计

账号安全服务提供云审计 CloudAudit 能力，可自动记录和存储腾讯云账号已执行操作的事件日志以及 API 调用日志，方便后续对用户行为进行回溯，简化您日后会进行的合规性审核。
