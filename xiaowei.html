功能介绍

  • 小微硬件开放平台
  • 小微Skill开放平台
  • 小微服务机器人平台
小微硬件开放平台

小微硬件开放平台给需要接入使用腾讯云小微语音能力的引荐厂商提供完整的对接流程和帮助，扩大腾讯云小微产品覆盖面。无论是音箱、电视、玩具、OTT盒子、投影仪还是汽车，只需要一个 SDK 即可完成接入。

小微硬件开放平台接入流程：

1. 申请接入
提交申请后工作人员会尽快反馈接入申请结果；

2. 植入SDK
通过接入申请后，进行 SDK 植入及接入调试；

3. 提交审核
接入调试完成后，申请进行产品认证；

4. 完成对接
产品认证通过后，就完成了正式产品对接，可以进行产品上线发布。

小微Skill开放平台

小微Skill开放平台开放语义分析能力给第三方，由小微提供设备控制、内容资源、服务能力。Skill平台包含腾讯自有音乐及各种有声音读物、新闻、笑话、天气等内容与服务，还可以让开发者创建自己的内容和服务。

小微Skill开放平台接入流程：

1. 申请接入
提交申请后工作人员会尽快反馈接入申请结果；

2. 植入SDK
通过接入申请后，进行 SDK 植入及接入调试；

3. 提交审核
接入调试完成后，申请进行产品认证；

4. 完成对接
产品认证通过后，就完成了正式产品对接，可以进行产品上线发布。

小微服务机器人平台

小微服务机器人平台，是为厂商的产品提供自动化售前售后咨询的一个智能机器人，减少客服人力成本。小微机器人就像是你的助理，开发者可以不断教她学习，在一定阶段后她可以帮助你做决策。

小微服务机器人能力介绍：

1. 客服不在线，支持全自动回复；

2. 客服上线后，消息对接人工客服；

3. 提供多客服同时运营服务；

4. 支持消息自动入库，重复问题推荐回答；

5. 支持公众平台和小程序以外的第三方接入，人工客服消息支持直接返回第三方消息系统。

 

应用场景

  • 家庭场景
  • 车载场景
  • 运动场景
家庭场景

家庭内带屏幕设备的接入可实现视频通话、交互可视化等功能。

车载场景

通过语音与汽车的结合，可实现语音控制导航、音乐播放、语音交互进行车内控制等操作。

运动场景

语音与运动耳机的结合，可实现耳机唤醒、连接手机APP完成互动等功能，让您在户外运动更方便自如。
