全球应用加速 GAAP 简介

全球应用加速（Global Application Acceleration Platform，GAAP）依赖全球节点之间的高速通道、转发集群及智能路由技术，实现各地用户的就近接入，通过高速通道直达源站区域，帮助业务解决全球用户访问卡顿或者延迟过高的问题。GAAP 提供图形化配置界面，只需几分钟，您就可以创建并使用高速通道访问您的业务源站，并通过控制台查看通道的运行情况。

腾讯云 GAAP 的特性

全球布局

全球拥有50+个节点组成高速通道网络，在各节点之间提供稳定高速的对等连接服务，为跨大区访问、全球同服的业务提供稳定高效的高速通道服务，解决业务访问卡顿和时延过高的问题。

稳定高效

五大中心点覆盖全球各大洲/区域，链路稳定性为99.95%；单通道支持1G带宽、100万并发的加速能力；通道组可同时管理多条通道，支持统一配置转发规则，支持各通道带宽合并计费。 

协议支持

通过在通道上创建监听器并配置路由转发规则，单条通道可配置多个监听器，同一个监听器可绑定多个源站。支持 TCP/UDP/HTTP/HTTPS 协议，支持端口映射、URL 规则转发。

快捷接入

腾讯云 GAAP 提供 IP 和域名两种接入方式。支持全球统一域名接入，实现多地域加速回源；提供源站负载均衡能力，支持3种转发调度策略；支持源站在腾讯云及腾讯云之外的云平台或自建 IDC 的业务接入。 

用户分析

源站可获取用户的真实 IP，GAAP 会把用户真实 IP 有效透传到源站，源站服务器只需按照 TOA 模块便可获取，提供 CentOS、Ubuntu、Debian 和 Windows 操作系统的安装支持。 

运维支持

提供 SLA 保障，支持按需、业务无感知对高速通道进行扩容，确保在线业务无需停服调整；提供多维度统计分析，如带宽、并发、丢包、延迟、包转发等数据统计；支持对通道、源站云监控告警。

跨区域访问加速

跨区域访问业务，通常会因为区域之间的网络状况不同而导致延时和丢包率增加，影响跨域访问业务的用户体验，通过全球应用加速，在加速区域（用户所在区域）与源站区域（服务器所在区域）之间创建高速通道，可有效降低网络时延和丢包率，保障网络的快速和稳定性，提升用户体验。

游戏全球同服

通常我们有些游戏项目是提供给全球或者某一个大区域的玩家同时接入，这样的场景通常会面临各区域网络差异而导致体验下降及影响游戏的公平性。全球应用加速服务支持多个区域使用统一的域名接入，实现各地区玩家就近接入高速通道，实现全球或同一大区用户同时加速，直接提升用户体验，确保游戏的公平性。
