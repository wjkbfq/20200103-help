工作空间介绍

工作空间是什么？

一个工作空间是一个虚拟计算单元，我们为每个工作空间提供了独立的存储、计算资源以及开发环境。在一个工作空间内您可以存放自己的项目代码，安装所需要的软件环境，运行或编译自己的项目。

如何创建工作空间？

前往 Cloud Studio 控制台（dashboard），点击「新建工作空间」的卡片就可以创建工作空间。

创建工作空间时，您有三个来源可以选择：腾讯云开发者平台（原 Coding.net）、其他 Git 仓库或者无。前两者会分别将您的工作空间绑定至腾讯云开发者平台和其他 Git 仓库，您可以通过 Git 同步代码，选择「无」则不绑定仓库。

如果选择从腾讯云开发者平台已有项目创建，在项目列表右上角有一个「同步」按钮，这是因为有些新创建的项目还没有同步过来，如果发现腾讯云开发者平台项目列表和这里的列表不统一，点击「同步」按钮即可。

当然您也可以选择从模板创建，「项目模板」是一系列打包好的运行环境和模板代码，您可以通过它们快速创建一些常用的项目。

如果您选择从其他 Git 仓库（如 Github）创建工作空间，你需要提前将 SSH Key 添加到来源 Git 仓库的公钥列表中（如何添加请参考这里），同时在仓库地址中填写 SSH 地址，这样新建的工作空间会将你仓库中的代码 clone 过来。

如何运行、停止、删除工作空间？

目前每个用户最多有 5 个工作空间，并且只能同时运行一个工作空间。点击工作空间的卡片就会在新页面中打开并运行该空间，此时你可以在控制台空间列表中点击「停止」按钮来停止它。如果您在一个工作空间运行时想要打开另一个工作空间，系统会提示您关闭当前运行中的空间。

不需要的工作空间可以删除，已删除的工作空间会展示在「最近删除」列表，保留 24 小时，在此之前您可以随时恢复，否则将被永远销毁。
