私有网络 VPC 简介

私有网络（Virtual Private Cloud，VPC）是基于腾讯云构建的专属云上网络空间，为您在腾讯云上的资源提供网络服务，不同私有网络间完全逻辑隔离。作为您在云上的专属网络空间，您可以通过软件定义网络的方式管理您的私有网络 VPC，实现 IP 地址、子网、路由表、网络 ACL 、流日志等功能的配置管理。私有网络还支持多种方式连接 Internet，如弹性 IP 、NAT 网关等，并提供多种计费方式和带宽包帮您节约成本。同时，您也可以通过 VPN 连接或专线接入连通腾讯云与您本地的数据中心，灵活构建混合云。

腾讯云 VPC 的特性

高可用

腾讯云采用成熟的网络虚拟化技术，通过链路冗余，网关主备，私有网络跨可用区部署等多种技术，保证网络高可用，从链路级，网络级，数据中心级全方位保障网络安全、稳定。

完全隔离

腾讯云通过 Overlay 技术，帮助您构建一个完全隔离的私有网络环境，不同私有网络间100%完全逻辑隔离，满足您业务的安全隔离需要。

多样化接入

腾讯云私有网络为您提供了多种形式的网络接入，您可以按需选择专线接入、VPN 连接等方式，连接您本地数据中心与腾讯云上的资源。

高性能 Internet

腾讯云支持超高性能的 Internet 访问，NAT网关支持 1000W 并发连接，帮助您轻松打破 Internet 访问性能瓶颈；并可搭配弹性负载均衡、Anycast 公网加速等产品提升业务访问体验。

多维度安全

腾讯云私有网络支持多维度的网络安全管控， 您可以通过网络 ACL 和安全组来实现端口维度和实例维度的资源访问控制；支持通过CAM实现账户最小授权，帮助您全方位提高网络安全性。

可视化管理

私有网络支持网络拓扑可视功能，可实时监控网络状态，设置流量告警，配合流日志服务，帮助您轻松实现网络全方位、智能化管理。

弹性扩展

腾讯云支持私有网络弹性扩容，无需采购、安装、调试等繁杂环节，让网络资源随着您的业务规模变化而灵活扩展，助力您业务规模增长。

成本最优

腾讯云私有网络大部分功能均可免费使用，您无需额外购买专用网络设备，即可享受高规格的数据中心级网络资源；同时腾讯云也为您提供多种计费方式和套餐包，供您灵活选择，实现最优配置。

服务集成

私有网络 VPC 与腾讯云的大部分业务都可以做到高度集成，比如云服务器CVM、云数据库 TencentDB、大数据处理套件 TBDS 等，为用户提供完整的云上解决方案。

云上高性能网络

大流量视频、直播类业务

构建云上互动直播平台，支持超100G海量视频实时传输，满足直播业务的一比多内网弹性转码需求，满足用户在不同场景下的使用体验。

  • 基于 VPC 构建高性能10Gbps云上网络，支持 1：n 视频转码；
  • 大带宽弹性 IP，支持 3G以上视频流量传输；
  • 基于共享带宽包，实现多 IP 共享带宽，降低成本。

云上物联网络

物联网业务

基于腾讯云构建的物联网服务架构，满足共享物联时代数十亿终端连接需求，如共享充电宝、共享单车、无人货架等物联场景。

  • CLB 支持海量并发，满足海量物联需求；
  • 35线 + BGP 线路满足任意地域物联终端接入；
  • NAT 访问互联网，保护内网资源安全。

云上业务加速

游戏加速

游戏客户业务遍布全球，不同区域部署多套服务器，通过对等连接，满足跨服 PVP 场景等网络时延敏感型业务，提升玩家体验。

  • 基于对等连接构建稳定、低时延网络，满足全球玩家同服竞技需求；
  • 全球支持众多地理区域和可用区，业务就近部署，满足不同玩家接入需求。

云上两地三中心

云上两地三中心

基于 VPC 构建云上两地三中心，保证云上金融业务安全、高可用，满足金融业务需求。

  • 业务跨可用区部署实现同城双活；
  • 对等连接实现异地数据灾备；
  • 基于 ACL、安全组、CAM 的多级安全防护实现业务区安全隔离 。

混合云架构

云上云下构建混合云

基于腾讯云构建云上多数据中心，通过专线/VPN 接入连接本地 IDC，满足数据云上灾备，弹性业务云上部署，构建混合云。

  • VPN/专线多链路接入；
  • 弹性业务云上部署，从容面对业务变化。
