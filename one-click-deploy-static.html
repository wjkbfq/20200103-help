使用一键部署搭建静态网站

文章内容

  • 1 创建应用，构建静态文件
  • 2 一键部署网站

一键部署可以帮你快速部署动态（PHP，支持 MySQL）或静态网站，如果您要部署动态网站请参考一键部署使用指南。下面我们将介绍如何通过一键部署来快速搭建一个静态站点，这里我们以 vuepress（一个用于构建静态文档网站的应用）为例。

创建应用，构建静态文件

首先，新建一个以 Node.js 为环境的工作空间。进入之后，我们先安装 vuepress，在终端执行以下命令：

# 安装
sudo yarn global add vuepress
# 或者
sudo npm install -g vuepress

安装成功之后，我们在当前目录下新建一个 README.md：

# 新建一个 markdown 文件
echo '# Hello VuePress!' > README.md

这时候我们就有了第一篇文档，我们执行命令来生成静态文件：

# 构建静态文件
sudo vuepress build .

此时，vuepress 将在 .vuepress/dist/ 下生成静态文件。

一键部署网站

现在已经有了静态文件，我们就可以将其一键部署到线上了。将静态文件目录 /.vuepress/dist 填写在「入口」中，点击右侧按钮保存入口：

点击下方的「重新部署」按钮，将网站部署上线。此时，点击测试域名就可以看到效果了。

我们也可以给网站绑定域名，详细操作方式请参考 https://dev.tencent.com/help/cloud-studio/one-click-deploy#i-4 。
